# 🌱 Agrinho 2026

## Tema

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.**

---

## Assunto Escolhido

**Energia Solar nas Propriedades Rurais**

---

## 📖 Sobre o Projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação Web.

O objetivo é apresentar como a energia solar pode transformar propriedades rurais, reduzindo custos com energia elétrica, preservando o meio ambiente e incentivando um modelo de produção agrícola mais sustentável.

O site foi desenvolvido utilizando HTML, CSS e JavaScript, oferecendo uma experiência interativa por meio de um quiz educativo sobre energia solar.

---

## 🎯 Objetivos

- Incentivar práticas sustentáveis no campo.
- Demonstrar a importância da energia solar na agricultura.
- Conscientizar sobre a preservação ambiental.
- Mostrar como tecnologia e sustentabilidade podem caminhar juntas.

---
